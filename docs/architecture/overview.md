# 架构概览

## 系统边界

```
┌─────────────────────────────────────────────────────────┐
│                    用户浏览器                            │
└─────────────────────────┬───────────────────────────────┘
                          │
                          ▼
┌─────────────────────────────────────────────────────────┐
│                    前端应用                              │
│  - 文章链接输入                                          │
│  - 解析结果展示                                          │
│  - 导出操作                                              │
└─────────────────────────┬───────────────────────────────┘
                          │ HTTP API
                          ▼
┌─────────────────────────────────────────────────────────┐
│                    后端服务                              │
│  - 链接解析                                              │
│  - 内容提取                                              │
│  - 图片处理                                              │
│  - 结果格式化                                            │
└─────────────────────────┬───────────────────────────────┘
                          │
                          ▼
┌─────────────────────────────────────────────────────────┐
│                  微信公众号页面                          │
│  - 获取文章HTML                                          │
│  - 提取图片资源                                          │
└─────────────────────────────────────────────────────────┘
```

## 核心模块

| 模块 | 职责 | 依赖 |
|-----|------|------|
| 解析器 | 解析公众号文章链接，提取HTML | requests/httpx |
| 内容提取器 | 从HTML中提取正文内容 | BeautifulSoup/lxml |
| 图片处理器 | 识别并处理图片URL | - |
| API服务 | 提供HTTP接口 | FastAPI/Flask |
| 前端界面 | 用户交互界面 | React/Vue |

## 主要数据流

```
用户输入链接
    │
    ▼
后端验证链接格式
    │
    ▼
请求公众号页面
    │
    ▼
解析HTML内容
    │
    ├─ 提取文章元信息
    ├─ 提取正文内容
    └─ 提取图片列表
    │
    ▼
格式化结果
    │
    ▼
返回给前端展示
```

## 技术选型

### 后端

- **语言**: Python
- **理由**: 丰富的HTML解析库，开发效率高

### 前端

- **框架**: 待定（React/Vue）
- **理由**: MVP阶段快速开发

### 部署

- **容器化**: Docker
- **理由**: 环境一致性，便于部署

## 约束与限制

- 需遵守微信公众号的robots协议
- 需处理反爬机制
- 图片资源可能有时效限制
- 部分文章可能需要登录才能访问
